# 🧪 اختبار سريع للمساعد الذكي

## ✅ قائمة اختبار بسيطة (5 دقائق)

### 1. التحقق من الظهور 👀

- [ ] شغّل `npm run dev`
- [ ] افتح الموقع على `http://localhost:3000`
- [ ] ابحث عن 🤖 في قائمة النشر
- [ ] يجب أن يكون في أعلى القائمة

### 2. فتح المساعد 🔌

- [ ] انقر على 🤖
- [ ] يجب أن يظهر مودال محادثة
- [ ] الرسالة الأولى: "مرحباً! ما نوع العنصر..."
- [ ] ثلاث خيارات ظاهرة

### 3. اختبار السيارة 🚗

```
الخيار 1: اختر "🚗 سيارة"
  [ ] تُضاف الرسالة للمحادثة
  [ ] السؤال التالي: "هل تريد بيع أم إيجار؟"

الخيار 2: اختر "🛒 بيع"
  [ ] لا يظهر سؤال rentType (صحيح)
  [ ] تسلسل الأسئلة صحيح

الخيار 3: اختر "🚗 إيجار"
  [ ] يظهر سؤال rentType (صحيح)
  [ ] الأسئلة الشرطية تعمل

الخيار 4: اكمل باقي الأسئلة
  [ ] كل سؤال له نوع إدخال صحيح
  [ ] النصوص تُحفظ
```

### 4. اختبار الصور 📸

- [ ] بعد انتهاء الأسئلة، تنتقل للصور
- [ ] منطقة رفع الصور مرئية
- [ ] زر "العودة" يرجع للمحادثة
- [ ] اختر صور من جهازك

### 5. الإرسال ✅

- [ ] اضغط "تأكيد"
- [ ] رسالة البحث تظهر
- [ ] رسالة النجاح تظهر
- [ ] المودال ينغلق تلقائياً

### 6. الأنواع الأخرى 🔄

- [ ] أعد المحادثة
- [ ] اختبر "🏠 عقار"
- [ ] اختبر "📦 منتج عام"
- [ ] كل نوع له أسئلته الصحيحة

## 🐛 إذا لم يعمل شيء

### المشاكل الشائعة:

**المساعد لا يظهر في القائمة:**

- [ ] تأكد من عمل الموقع بدون أخطاء
- [ ] افتح console بـ F12
- [ ] ابحث عن أخطاء بـ "ChatAssistant"

**المحادثة لا تستجيب:**

- [ ] أعد تحميل الصفحة
- [ ] امسح cache المتصفح
- [ ] تأكد من حفظ الملفات

**الصور لا ترفع:**

- [ ] تأكد من اختيار صورة واحدة على الأقل
- [ ] تحقق من حجم الملف
- [ ] راجع Network tab في Developer Tools
