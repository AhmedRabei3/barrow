# Smart Chat Assistant - دليل الاستخدام

## نظرة عامة

تم استبدال طريقة إضافة العناصر التقليدية بمساعد ذكي يوجه المستخدم عبر محادثة تفاعلية.

## الميزات

### 1️⃣ محادثة تفاعلية

- المساعد يطرح الأسئلة خطوة بخطوة
- واجهة محسّنة مع رسائل متحركة
- إدخال متكيف حسب نوع السؤال (أزرار، نص، أرقام، ألوان، نصوص طويلة)

### 2️⃣ أسئلة شرطية (Conditional Questions)

```
مثال: إذا اختار المستخدم "إيجار" للسيارة، يظهر السؤال عن فترة الإيجار (يومي/أسبوعي/شهري)
```

### 3️⃣ دعم ثلاث أنواع من العناصر

- 🚗 **السيارات**: ماركة، موديل، سنة، لون، وقود، ناقل حركة، سعر، نوع الإيجار
- 🏠 **العقارات**: نوع العقار، المساحة، عدد الغرف، الحمامات، السعر

- 📦 **منتجات عامة**: الاسم، السعر، الوصف
